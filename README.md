# Spam-Ham Classifier 

A simple ML project that classifies messages as **Spam** or **Ham (Not Spam)**.  
Built with Python, pandas, and scikit-learn. Used simple NLP tactics and text classification.  


## Features

- Cleaned and preprocessed text data  
- Trained a spam detection model  
- Predicted whether a message is spam or not  
- Can extend with advanced models
